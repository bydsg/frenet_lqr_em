# frenet_lqr_em<br>
使用方法<br>
roslaunch by_djstl all.launch  <br>
(launch里面包括by_djstl启动全局路径规划发送给two_by二次优化全局路径的参考线发送给la驱动frenet采样规划，控制方式有两种pure_pursuit_by下的纯追踪和lqr_by下的lqr（有bug不想调了，用纯追踪挺好的）)<br>
